# ATM Interface (Core Java)

**Student:** K Navya  
**Domain:** CSE/ Java  
**Task:** OIBSIP_IT_ATMInterface_Task03

## Objective
Build a console-based ATM interface to simulate real ATM operations and strengthen OOP and exception handling concepts.

## Features
- PIN authentication
- Check balance
- Deposit amount
- Withdraw amount (with balance checks)
- Transaction history (session-based)
- Menu-driven interface

## Tools & Language
- Java (JDK 11+)
- IDE: VS Code / IntelliJ IDEA (optional)

## How to run
1. Compile:
   `javac ATMInterface.java` (or compile all files in `cmd`)
2. Run:
   `java ATMInterface`
